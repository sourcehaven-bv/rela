---
id: BUG-006
status: done
title: Dark mode link color not legible
type: bug
description: |
  Link colors in dark mode used #60a5fa (blue-400) which had insufficient
  contrast against the dark background (#0f172a). This made links in tables,
  markdown content, and other UI elements difficult to read.
priority: low
effort: xs
why1: The --primary CSS variable was set to #60a5fa in dark mode, which has low contrast
why2: The dark mode color values were chosen without checking WCAG contrast ratios
why3: No visual accessibility testing was performed for dark mode
prevention: |
  Use a color contrast checker when selecting dark mode colors. The fix uses
  #93c5fd (blue-300) which provides better contrast against dark backgrounds.
---
