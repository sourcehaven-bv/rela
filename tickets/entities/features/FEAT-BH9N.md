id: FEAT-BH9N
type: feature
title: Keyboard delete shortcut in data-entry lists and detail view
summary: Allow users to delete entities via keyboard shortcut from list views (selected row) and from the entity detail view, always behind a confirmation modal.
description: 'Power-users navigate the data-entry app heavily by keyboard. This feature adds a `Delete` keyboard shortcut in two places: (1) on list/table views, deleting the currently-selected row, and (2) on entity detail view, deleting the displayed entity. Both paths always show a confirmation modal before performing the delete — destructive actions are never one-keystroke.'
priority: medium
status: proposed
